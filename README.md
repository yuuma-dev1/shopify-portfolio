# Shopifyポートフォリオ（ECサイト構築デモ）

植物由来のヘアケアブランドを想定した、ShopifyのECサイトデモです。

## デモストア

URL：  
https://ki-to-ka.myshopify.com

閲覧パスワード：  
maifro

## 概要

ナチュラルで清潔感のあるヘアケアブランドを想定し、商品一覧・商品詳細・カートまでの基本的な購入導線を確認できるECサイトとして制作しました。

ユーザーが迷わず商品を探し、購入まで進めるように、シンプルなメニュー構成と見やすいレイアウトを意識しています。

## 対応した内容

- 商品登録
- 商品一覧ページの作成
- 商品詳細ページの作成
- カート機能の確認
- ナビゲーション設定
- トップページのレイアウト調整
- ロゴ・カラー・フォントの調整
- モバイル表示の確認

## 意識した点

- ブランドイメージに合う落ち着いた配色
- 商品にたどり着きやすい導線設計
- 余白を活かした見やすいレイアウト
- スマホでも見やすいシンプルな構成

## 使用ツール

- Shopify
- Shopifyテーマカスタマイズ
- ノーコード編集
